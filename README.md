# @zammit/eslint-config-az

Lints es2021 modules or scripts, yaml, and jsdoc.


## Installation

```
yarn add --dev \
  @zammit/eslint-config-az \
  eslint \
  eslint-plugin-import \
  eslint-plugin-jsdoc \
  eslint-plugin-unicorn \
  eslint-plugin-yml

printf '%s\n' 'extends: \'@zammit/eslint-config-az\'' > .eslintrc.yml
```
